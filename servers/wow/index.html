World of Warcraft Server

Access Details

  1. Aquire a copy of World of Warcraft: The Burning Crusade (v2.4.3)
  2. Open your install folder
  3. Edit realmlist.wtf and set both realmlist and patchlist to: wow.pez.solutions

Account Creation

Please request an account by electronic mail on pez@pez.sh

Setup

This server is setup as a vanilla server. XP rates is set to 1x of the original game.

The server is hosted in the Copenhagen location.
