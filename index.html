Connecting to Pez Solutions Services

Plex Access

Step 1 - Creating a Plex Account

  1. Go to the Plex website at https://plex.tv.
  2. Click on the "Sign Up" button in the top right corner of the page.
  3. Enter your email address, username, and password to create an account.
  4. Click on the "Create Account" button.

Step 2: Requesting Access to Pez Solutions Plex Server

  1. Send an email to plex@pez.solutions with your Plex username.
  2. Request to be added to the Pez Solutions Plex server.
  3. Wait for a response from the Pez Solutions team.

Step 3: Waiting for Access Approval

  1. After about 1 hour, your account will be added to the Pez Solutions Plex server.
  2. You will receive an email notification confirming your access.

Step 4: Accessing the Pez Solutions Plex Server

  1. Open the Plex app on your device.
  2. Click on the "Servers" tab.
  3. Select the "Pez Solutions" server from the list.
  4. You will now have access to the Pez Solutions Plex server and can start streaming content.

Note: Please note that the Pez Solutions Plex server is a private server and access is restricted to authorized users only. Attempting to access the server without proper authorization may result in a ban

Jellyfin Access

Step 1 - Request Access

To request access to a Jellyfin server, please send an email to jellyfin@pez.solutions with the following information:

  • Your name and affiliation
  • Your desired username for the Jellyfin server

Please note that requests are typically responded to within 1 hour.

Step 2 - Receiving Your Autogenerated Password

Once your request has been reviewed, you will receive an email with an autogenerated password for the Jellyfin server. This password can be used to log into the server for the first time.

Step 3 - Logging into Jellyfin

To log into the Jellyfin server, please visit the following URL: jellyfin.pez.solutions

Enter your generated password in the password field and click the "Log In" button.

Step 4 - Changing Your Password (Optional)

Once you have successfully logged into the Jellyfin server, you can change your password by clicking on the "User Settings" link in the top right corner of the page. From there, you can update your password to something more memorable.

That's it! You should now have access to the Jellyfin server and be able to use its features and functionality. If you encounter any issues or have questions, please don't hesitate to reach out to the Jellyfin support team.
